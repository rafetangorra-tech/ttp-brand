## For Contributors

If you're adding a new asset, follow these rules:

- Place files in the matching `assets/` subfolder.
- Use lowercase-kebab-case filenames (`logo-primary.png`, not `Logo Primary.PNG`).
- Include source files (`.ai`, `.psd`, `.fig`) when available alongside exports.
- Update the relevant `brand/*.md` doc if you add new variants.
- Don't commit third-party fonts — link to them (Google Fonts, Adobe, etc.).
