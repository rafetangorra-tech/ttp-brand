# Logo Usage

## Files

| File | Use Case |
|---|---|
| [`assets/logo/logo-primary.png`](../assets/logo/logo-primary.png) | Primary header logo (default usage) |
| [`assets/logo/logo-footer.png`](../assets/logo/logo-footer.png) | Footer / dark-background lockup |
| [`assets/logo/logo-mark.png`](../assets/logo/logo-mark.png) | Square mark (profile avatars, app icons) |
| [`assets/favicon/`](../assets/favicon/) | Favicon + touch icon set (32, 180, 192, 270) |

> **Missing:** Vector (`.svg`, `.ai`, `.eps`) source files. These should be requested from the original designer and added to this repo — PNGs alone don't scale for print or large signage without quality loss.

## Clear Space

Maintain clear space equal to the height of the "T" in "Trainer" on all sides of the logo. Nothing — text, imagery, UI chrome — enters that buffer.

## Minimum Size

- **Digital:** 120px wide for the primary lockup; 32px for the square mark.
- **Print:** 1 inch wide for the primary lockup; 0.5 inch for the square mark.

## Backgrounds

| Background | Which logo? |
|---|---|
| White / light | Primary (dark) logo |
| TTP Red `#C60101` | Footer / white version |
| TTP Navy `#112337` | Footer / white version |
| Photography | Use white version; add a subtle dark gradient if needed for contrast |

## Don'ts

1. Don't recolor the logo outside of the approved palette.
2. Don't stretch, squash, or rotate.
3. Don't add drop shadows, outlines, or effects.
4. Don't place on busy imagery without contrast treatment.
5. Don't reconstruct the wordmark in another typeface.
6. Don't crop the mark.
7. Don't use low-resolution exports — request vector sources.
